#Cell relaxation constraint type (datatype: integer, range: 1-12, default: 12)
# 1 (isotropic shape fixed volume optimization) 
# 2 (volume fixed shape optimization) 
# 3 (relax only cell component v1_x) 
# 4 (relax only cell component v2_x)
# 5 (relax only cell component v3_x)
# 6 (relax only cell components v2_x and v3_x) 
# 7 (relax only cell components v1_x and v3_x)
# 8 (relax only cell components v1x and v2_x) 
# 9 (volume optimization- relax only v1_x, v2_x and v3_x) 
# 10 (2D- relax only x and y components relaxed) 
# 11 (2D- relax only x and y shape components- inplane area fixed)
# 12 (relax all cell components)
CELL CONSTRAINT TYPE
